# Movie Catalog

## Description

Movie Catalog is a Django web application that allows users to:

- Register and log in
- Browse movies
- View movie details
- Add new movies
- Write reviews
- View reviews from other users

## Technologies

- Python
- Django
- Django REST Framework
- HTML
- CSS
- SQLite

## Features

- User authentication
- Movie catalog
- Movie reviews
- REST API
- Django Templates
